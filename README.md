# 💌 Love Letter — A Digital Gift

A fullscreen, animated love letter website with 8 beautifully animated pages, custom voice note player, and smooth scroll-based navigation.

## ✨ Features

- **8 animated pages** — each with layered animations over PNG backgrounds
- **Smooth page transitions** — Framer Motion crossfade between pages
- **Voice note player** — real-time audio waveform visualizer using Web Audio API
- **Multiple navigation** — scroll, swipe, keyboard arrows, and navigation dots
- **Mobile responsive** — optimized for all screen sizes
- **Loading screen** — animated heart while assets preload

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📂 Assets

Place your assets in `/public/assets/`:

| File | Description |
|------|-------------|
| `cover.png` | Cover page (Just For You folder) |
| `intro.png` | Introduction page |
| `about-you.png` | About You page |
| `memory-lane.png` | Memory Lane Moments |
| `cherished.png` | Cherished Memories |
| `special.png` | What Makes You Special |
| `celebrating.png` | Celebrating This Special Day |
| `voice-note.mp3` | Your voice note audio |
| `couple-photo.jpg` | Photo for voice note player |

## 🎨 Tech Stack

- **React 18** + **Vite**
- **Framer Motion** — page transitions and element animations
- **Tailwind CSS** — utility-first styling
- **Web Audio API** — real-time audio waveform visualization

## 🌐 Deployment

Deploy to Vercel:

```bash
npx vercel
```

Or connect your GitHub repo to [vercel.com](https://vercel.com) for automatic deployments.

## 🎯 Navigation

| Method | Action |
|--------|--------|
| **Scroll** | Mouse wheel to go up/down |
| **Swipe** | Touch swipe on mobile |
| **Keyboard** | ↑↓ or ←→ arrow keys |
| **Dots** | Click navigation dots on right side |
| **Click** | Click cover page to open |
